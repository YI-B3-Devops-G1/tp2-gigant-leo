## Installation

- Cloner le projet
```
    git clone git@github.com:YI-B3-Devops-G1/tp2-gigant-leo.git
```
- Entrez dans le terminal :
```
    cd tp2-gigant-leo
```
Puis :
```
    npm install
```
- Entrez enfin : 
```
    docker-compose up
```
- Vérifiez que le serveur web est accéssible et fonctionnel : http://localhost:8080 et http://localhost:8080/status
